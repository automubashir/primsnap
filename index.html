Why PrimSnap?

Built for developers who need accurate, reliable screenshots without the hassle.

Pixel-Perfect Accuracy

SVG foreignObject rendering ensures your screenshots match exactly what you see in the browser.

Full CSS Support

Gradients, transforms, filters, clip-paths, pseudo-elements, CSS variables - all captured perfectly.

Print Styles

Respects @media print CSS rules just like the browser's native print function.

Header & Footer

Add custom headers and footers with template variables like page numbers and dates.

Page Breaks

Control page breaks with break-inside: avoid for clean multi-page documents.

CSS/JS Injection

Inject custom styles or run JavaScript before capture for dynamic modifications.

Shadow DOM

Full support for Web Components and Shadow DOM traversal.

Web Fonts

Waits for fonts to load and inlines them as data URIs for reliable text rendering.

Zero Dependencies

Single file, no build tools, no external dependencies. Just include and use.

Simple API

JavaScript 
// Simple capture
const dataUrl = await PrimSnap.capture('#element');

// Download as PNG
await PrimSnap.download('#element', 'report.png');

// Preview in modal
await PrimSnap.preview('#element');

// With all options
await PrimSnap.capture('#element', {
  format: 'png',
  scale: 2,
  backgroundColor: '#ffffff',
  usePrintStyles: true,

  // Header/Footer
  showHeader: true,
  header: '<span>My Report</span><span>{date}</span>',
  showFooter: true,
  footer: '<span>Confidential</span><span>Page {page}</span>',

  // Page breaks
  avoidBreakInside: ['.section', '.card'],

  // CSS/JS injection
  injectCSS: '.no-print { display: none; }',
  injectJS: (el) => { el.querySelector('.date').textContent = new Date(); },

  // Callbacks
  onProgress: (p, msg) => console.log(p * 100 + '%')
});
